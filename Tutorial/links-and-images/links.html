this page is about links! there are three kinds of links:

  1. Absolute Links, like to Mozilla Developer Network, which is a very good resource for web developers.
    • These are best to direct users to a different website
    • Add the target='_blank' attribute to open the link in a new tab
  2. Relative Links, like to our extras page.
    • These are best when you are referring to resources in the same folder or in a stand alone section of your website
    • Pay close attention to the folder you are telling it to look in!
  3. Root-Relative Links, like to the home page of our website, butt those aren't useful to us right now.
    • The only different between a root-relative link and a relative link is that a root-relative link starts with a forwad slash which represents the root of your site.
