Casey's Proposal

Name the sites where you can discover where trivia night is happening in Pittsburgh... I'll wait.

My proposal is to design a website that acts as an intelligent and responsive database where users can look up trivia nights happening in Pittsburgh.

What will it look like?

The Landing Page

The main page will be simple and consist of a header and four buttons. Each button will refer to a different way to search for a trivia night. For example, by:

  • Location
  • Day of the Week
  • Theme
  • Drink Special

Inner Pages

The buttons on the front page will lead to the corresponding page which will feature the data arranged according to that specific attribute.

In total, I see this website containing 5 pages overall. But that is not set in stone, of course.

What is Involved?

  1. Research: We will need to collect the necessary data to build our database
  2. HTML: To build the structure of the webpage
  3. CSS: To design the style of the webpage

Any of these task can be cone by all members of the group if the work flow is organized well
