Night Search Engine
Name the sites where you can discover where trivia night is happening in Pittsburgh... I'll wait.
My proposal is to design a website that acts as an intelligent and responsive database where users can look up any trivia night that is happening in Pittsburgh.
What will it look like?

The main page will be simple and consist of a header and four buttons. Each button will refer to a different way to search for a trivia night. For example, by:

  • Location
  • Day of the Week
  • Theme
  • Drink Special

The buttons on the front page will lead to the corresponding page which will feature the data arranged according to that specific attribute. In total, I see this website containing 5 pages overall. But that is not set in stone, of course.

What needs to happen?
  • Research: We will need to collect the necessary data to build our database
  • HTML: To build the structure of the webpage
  • CSS: To design the style of the webpage
Any of these task can be done by any and all members of the group, so long as it is coordinated.
